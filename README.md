# Attendance System Computer Science Project

## Steps to run
1. Clone the repository
2. Restore NuGet packages
````
dotnet restore
````
3. Build the project
````
dotnet build
````
4. Run the project
````
dotnet run
````
5. Navigate to the AttendanceSystem folder
````
cd AttendanceSystem
````
6. Open the browser and navigate to `http://localhost:5000`
